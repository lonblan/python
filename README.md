Para crear carpeta o archivo desde Visual Code, procedemos así :

1. Creamos el archivo o carpeta y se creará inmediatamente en la carpeta local.

2. En la parte de Source Control donde figura el archivo pendiente por actualizar, hacemos click para que se actualice el archivo y se genere un commit.

3. En Source Control se queda una especie de reloj pendiente. damos click y seguimos con el commit si es necesario.

4. YA los elemento tanto en Visual como en la carpeta local están actualizados.

5. PAra eliminar los archivos, damos click derecho y lo eliminamos. En la carpeta local también se eliminarán.
